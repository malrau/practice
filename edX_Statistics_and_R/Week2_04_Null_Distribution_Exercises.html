Null Distribution Exercises


For these exercises, we will use the data set of the entire mice control population. We load it and unlist it, in order to deal with a numerical vector. control_pop represents the weights for the entire mice population.

Null Distribution Exercises #1

1 point possible (graded)

Set the seed at 1, then using a for-loop take a random sample of 5 mice 1000 times. Save these averages. What proportion of these 1000 averages are more than 1 gram away from the average of control_pop?

## [1] 0.503
The result is wrong.
However, setting the seed at 1 returns this value (The accepted result is: 0.498)

Null Distribution Exercise #3

1 point possible (graded)

Q:Note that the answers to 4 and 5 barely changed. This is expected. The way we think about the random value distributions is as the distribution of the list of values obtained if we repeated the experiment an infinite number of times. On a computer, we cant perform an infinite number of iterations so instead, for our examples, we consider 1000 to be large enough, thus 10000 is as well. Now if instead we change the sample size, then we change the random variable and thus its distribution.

Set the seed at 1, then using a for-loop take a random sample of 50 mice 1000 times. Save these averages. What percent of these 1000 averages are more than 1 ounce away from the average of control_pop ?

## [1] 0.014
The result is wrong.
However, setting the seed at 1 returns this value (The accepted result is: 0.019)
